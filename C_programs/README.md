
# C Programs

This folder contains basic C programs written for practice and laboratory work.
The programs mainly cover array operations, searching, and sorting techniques.

## List of Programs
1. array_1.c – Basic array operations
2. array_2.c – Array traversal
3. arry_average.c – Find average of array elements
4. binary_search.c – Binary search algorithm
5. bubble_sort.c – Bubble sort algorithm
6. copy_array.c – Copy elements from one array to another
7. count_frequency.c – Count frequency of elements in array
8. insertion_sort.c – Insertion sort algorithm
9. linear_search.c – Linear search algorithm
10. max_min.c – Find maximum and minimum elements
11. number_check.c – Check number properties
12. remove_duplicate.c – Remove duplicate elements from array
13. reverse_array.c – Reverse array elements
14. selection_sort.c – Selection sort algorithm
15. sumofelements.c – Sum of array elements

## How to Compile and Run
Use the following commands:

gcc filename.c  
./a.out
